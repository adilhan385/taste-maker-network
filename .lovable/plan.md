

# План: Компактная админ-панель

## Проблема
Боковая панель с 8 пунктами не вмещается, особенно на средних экранах и с учётом кнопок действий внутри табов.

## Решение
Заменить боковой сайдбар на **горизонтальную полосу табов** (как в браузере) вверху контента. На мобильных — горизонтальный скролл.

### Дизайн
- Убрать `w-64 bg-card border-r` сайдбар
- Вместо него — горизонтальная полоса иконок + текст под заголовком страницы
- На десктопе: все табы видны в ряд (компактные кнопки с иконкой + текст)
- На мобильных: горизонтальный скролл (`overflow-x-auto`)
- Кнопка "Выйти" переносится в заголовок (справа)

```text
┌──────────────────────────────────────────────────┐
│  🛡 Админ-панель          [Поиск...]    [Выйти] │
├──────────────────────────────────────────────────┤
│ [👨‍🍳Заявки] [👥Юзеры] [📦Товары] [💳Заказы]...│
├──────────────────────────────────────────────────┤
│                                                  │
│              Контент активного таба               │
│                                                  │
└──────────────────────────────────────────────────┘
```
