

# План: Исправить отображение длинных сообщений в чате

## Проблема
Когда сообщение содержит много символов без пробелов (или просто очень длинный текст), оно выходит за границы пузырька и ломает layout чата.

## Причина
В `src/pages/Chat.tsx` (строка 403) у пузырька сообщения есть `max-w-[70%]`, но нет CSS для переноса длинных слов. Длинные строки без пробелов не переносятся и выходят за пределы контейнера.

## Исправление

### `src/pages/Chat.tsx`
- Добавить `break-words overflow-hidden` к div пузырька сообщения (строка 403)
- Добавить `break-all whitespace-pre-wrap` к тексту `<p>` (строка 404)

Итоговый класс пузырька:
```
max-w-[70%] rounded-2xl px-4 py-2 break-words overflow-hidden
```

Итоговый класс текста:
```
text-sm whitespace-pre-wrap break-words
```

Также исправить аналогичное место в `AdminChatsTab.tsx` (строка ~130) — там тоже пузырьки без переноса.

| Файл | Действие |
|------|----------|
| `src/pages/Chat.tsx` | Добавить break-words к пузырькам |
| `src/components/admin/AdminChatsTab.tsx` | Добавить break-words к пузырькам |
