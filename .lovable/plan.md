

# План: Добавить кнопку выдачи/снятия прав админа

## Что будет сделано

В таб "Пользователи" админ-панели добавить кнопку "Сделать админом" / "Снять админа" для каждого пользователя (кроме себя).

## Изменения

### `src/components/admin/AdminUsersTab.tsx`
- Добавить кнопку `Shield` рядом с кнопками бана/ранга
- Для пользователей с ролью != 'admin': кнопка "Сделать админом" — `INSERT INTO user_roles (user_id, role) VALUES (x, 'admin')`
- Для пользователей с ролью 'admin' (кроме текущего): кнопка "Снять админа" — `DELETE FROM user_roles WHERE user_id = x AND role = 'admin'`
- Добавить подтверждение через диалог перед назначением/снятием
- Перезагрузка списка после действия

### `src/lib/i18n.ts`
- Добавить ключи: `admin.makeAdmin`, `admin.removeAdmin`, `admin.confirmMakeAdmin`, `admin.confirmRemoveAdmin`

## Как выдать сейчас (до реализации)

Пока функция не реализована в интерфейсе, выдать админку можно только через базу данных напрямую — это невозможно из UI. После реализации — кнопка появится в списке пользователей.
